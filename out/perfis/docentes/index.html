Perfis e ResponsabilidadesDocentes

Docentes

Os docentes são os responsáveis por fornecer dados brutos de uma ata, especificamente o parecer sobre as turmas pertinentes às matérias que lecionam. Este perfil possui acesso de visualização e acompanhamento.

Responsabilidades Principais

Visualização de Dados

  • Visualizar dados das atas relacionadas às suas disciplinas
  • Consultar informações de turmas onde leciona
  • Identificar coordenadores das turmas
  • Ver outros docentes que lecionam nas mesmas turmas

Acompanhamento de Atas

  • Visualizar atas na íntegra que são pertinentes ao docente
  • Acompanhar atas em andamento relacionadas às suas turmas
  • Consultar atas finalizadas para referência

Informações de Turmas

  • Listar turmas onde atua como docente
  • Ver coordenador responsável por cada turma
  • Consultar outros docentes da mesma turma

Funcionalidades Disponíveis

  • Visualização de dados das atas
  • Consulta de turmas e coordenadores
  • Acompanhamento de atas pertinentes
  • Visualização completa de atas finalizadas
  • Preencher parecer do docente em atas das turmas pertinentes

Fluxo de Trabalho Típico

  1. Acesso ao Sistema:

    • Login com credenciais fornecidas pela CSP
    • Navegação pela dashboard

  2. Localização de Atas:

    • Buscar atas na dashboard principal
    • Acessar pela seção de turmas

  3. Acompanhamento:

    • Verificar status das atas
    • Consultar informações das turmas
    • Visualizar atas completas quando necessário
    • Preencher parecer do docente quando pertinente

Onde Encontrar Informações

Dashboard Principal

  • Atas pendentes relacionadas às suas disciplinas
  • Atas finalizadas recentes
  • Resumo das turmas

Seção de Turmas

  • Lista completa das turmas onde leciona
  • Coordenadores responsáveis
  • Outros docentes da turma
  • Atas associadas a cada turma

Pontos de Atenção

  • Não possui permissão para editar informações, apenas campos de parecer do docente
  • Acesso limitado apenas às atas pertinentes

Visualização de Atas

Os docentes podem visualizar atas nos seguintes formatos:

  • PDF: Para impressão e arquivamento
  • DOCX: Para referência em documentos Word
  • Visualização online: Interface do sistema

Para instruções detalhadas, consulte o Treinamento para Docentes.
